Question-1: What are the different ways to select an element in the DOM?

There are several ways to select elements in the DOM. The easiest way to access a single element in the DOM is by its unique ID. Get an element by ID with the getElementById() method of the document object.

Question-2: What is the difference between innerHTML, innerText, and textContent?

In HTML, innerHTML, innerText, and textContent are properties of the DOM (Document Object Model). They allow you to read and update the content of HTML elements.

  • innerHTML: Gets or sets HTML content, including tags.
  • innerText: Gets or sets visible text, considering styles and layout.
  • textContent: Gets or sets all text content, ignoring styles and layout.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach a single event listener to a parent element instead of each child element. The event bubbles up, and the parent handles it, making it efficient and useful for dynamic content.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts at the target element and then bubbles up to its parent elements. This allows parent elements to handle events that occur on their child elements, reducing the need for adding separate event listeners to each child.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create an element, use document.createElement(tagName). To add it to the DOM, use appendChild(). To remove an element, use removeChild() or remove().
